# Log de Alterações - 27/06/2024

## Inicialização do Projeto
- Criação do projeto Next.js com TypeScript, Tailwind CSS e App Router
- Instalação de dependências: Prisma, bcryptjs, jose
- Configuração do Prisma com SQLite
- Definição do schema do banco de dados

## Estrutura Criada
- Models: UserAdmin, Question, Challenge, Game, Team, GameAction
- API Routes: auth (login, logout, me), admin (perguntas, provas, usuarios, games)
- Páginas: Home, Admin (login, dashboard, perguntas, provas, usuarios), Apresentador, Telão
- Middleware de proteção das rotas /admin
- Sistema de autenticação JWT com bcrypt
- Seed com 100 provas do Paga e admin padrão

## Configurações
- .env com DATABASE_URL, JWT_SECRET e MAGIC_WORD
- AGENTS.md com regras de desenvolvimento
- Documentação inicial em /docs
- README.md na raiz

## Design
- Tema cartunesco infantil (cores vibrantes, fontes Fredoka + Nunito)
- Fundo degradê laranja/quente nas páginas públicas
- Cards arredondados com bordas coloridas
- Botões com efeito 3D (border-bottom + active translate)
- Telão com fundo gradiente roxo/azul escuro
- Emojis como ícones em toda a interface

## Fluxo do Telão
- Etapa 1: Palavra cabalística (validada via API com MAGIC_WORD do .env)
- Etapa 2: PIN da partida (validado no banco de dados)
- Etapa 3: Tela do jogo com placar

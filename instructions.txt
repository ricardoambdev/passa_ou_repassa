# Projeto: Passa ou Repassa Web - Next.js + SQLite (Versão Completa com Área Administrativa)

Criar uma aplicação web inspirada no programa Passa ou Repassa, com:

- Tela do Apresentador
- Tela Telão
- Sistema de partidas
- Banco de perguntas
- Banco de provas do PAGA
- Área administrativa protegida por e-mail e senha


Tecnologias:

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Prisma ORM
- SQLite
- Autenticação com NextAuth ou sistema próprio com JWT
- bcrypt para criptografia de senha


---

# Estrutura do sistema

A aplicação terá três ambientes:


## 1 - Apresentador

Responsável pelo controle do jogo.


## 2 - Telão

Tela pública para jogadores.


## 3 - Administrativo

Área restrita para gerenciamento de conteúdo.
